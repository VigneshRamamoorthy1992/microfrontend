# React and Angular single-spa example

## Running the app
#### Clone the code 
```bash
git clone https://github.com/VigneshRamamoorthy1992/microfrontend.git && cd react-angular-single-spa
```

#### Install dependencies
```bash
npm install
```

#### Run app
```bash
npm start
```

#### Open `http://localhost:9090` in browser
